# Smart Lead Management System - Salesforce LWC Project

## Overview
This project automates lead creation, assignment, and tracking using Salesforce LWC and Apex.

## Features
- Lead creation via Lightning Web Component
- Apex-based backend logic
- Duplicate lead prevention using triggers
- Automated workflows using Flow

## Tech Stack
- Salesforce Apex
- Lightning Web Components (LWC)
- Salesforce Flows
